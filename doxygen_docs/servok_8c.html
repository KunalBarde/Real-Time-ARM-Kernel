Detailed Description

Servo controller implementation. Uses manual pwm generated by systick and implements two system call for enabling and disabling servos.

Date
Author

Function Documentation

int sys_servo_enable ( UNUSED uint8_t  channel,
UNUSED uint8_t  enabled 
)

Implementation of system call for servo enable.

Parameters
channelServo channel to enable (1 or 0).
enabledFlag to indicated whether to enable to disable the servo. Servo will be disabled if enabled == 0 and enabled otherwise.
Returns
0 on success, -1 on failure (most likely for requesting an incorrect channel).

UNUSED uint8_t  angle 
)

Implementation of system call for servo set.

Parameters
channelServo channel to set (1 or 0).
angleAngle to set the channel to.
Returns
0 on success, -1 on failure if the servo channel is not enabled.
