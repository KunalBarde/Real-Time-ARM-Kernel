sbrk system call implementation. Attempts to increase available heap size by an increment.

Parameters
incrIncrement of bytes by which to increase the heap.
Returns
(void*)-1 if the heap cannot be extended by the requested increment, otherwise a void pointer to the newly designated memory of size incr.
